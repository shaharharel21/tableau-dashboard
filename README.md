# Tableau Data Viz Dashboard 📊

A sleek, custom-designed web interface to showcase an interactive Tableau data visualization. Built with an emphasis on clean aesthetics, a modern dark-mode UI, and seamless Tableau Public integration.

🔗 **[View Live Demo](https://shaharharel21.github.io/tableau-dashboard/)**

## ✨ Features

* **Interactive Data:** Fully embedded Tableau Public dashboard utilizing the official Tableau API.
* **Custom UI/UX:** * Modern dark theme with glowing accents and an ambient background grid.
  * Custom loading spinner that smoothly transitions out once the visualization is fully loaded.
  * Dynamic live clock tracking the current time.
* **Immersive Viewing:** Built-in Full-Screen mode toggle for distraction-free data exploration.
* **Responsive Design:** Fluid layout that adapts to both desktop and mobile viewing experiences.
* **RTL Support:** Fully structured and localized in Hebrew (Right-to-Left layout).

## 🛠️ Tech Stack

* **Frontend:** HTML5, Vanilla CSS (CSS Variables, Grid, Animations), Vanilla JavaScript
* **Data Visualization:** Tableau Public API (`viz_v1.js`)
* **Fonts:** Google Fonts (Frank Ruhl Libre, Heebo, JetBrains Mono)

## 🚀 Getting Started

Since this is a static, front-end-only project, no complex setup is required:

1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser.
3. *Note: An active internet connection is required to load the Tableau API and fetch the dashboard data.*
